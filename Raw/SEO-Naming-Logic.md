# SEO Product Design Naming Logic

## Purpose
This document defines the naming conventions for SEO-optimized product design titles at Head Case Designs. Use these rules when generating or reviewing design names for marketplace listings.

---

## General Principles

1. **Use full, unabbreviated names** — never shorten brand or club names (e.g., "Manchester City FC" not "Man City", "Newcastle United FC" not "Newcastle FC").
2. **Front-load the brand/license** — the most searched term (club, franchise, character) should appear first.
3. **Keep titles concise but descriptive** — aim for 3–6 words. Every word should add search value.
4. **No internal codes or jargon** — titles are customer-facing. Never include SKU fragments or internal abbreviations.

---

## Category-Specific Rules

### Football / Soccer Kits
- **Format:** `[Full Club Name] [Year] [Home/Away/Third] Kit`
- Always include the kit year (e.g., "2025").
- Always specify Home, Away, or Third.
- Use "FC" or "CF" as the club officially uses it (e.g., "Real Madrid CF", "Arsenal FC").
- **Example:** `Arsenal FC 2025 Away Kit`

### Football / Soccer Non-Kit Designs (Crests, Art, Patterns)
- **Format:** `[Full Club Name] [Design Type] [Descriptor]`
- Design types include: Crest, Mural, Urban, Retro, Badge.
- Add a visual descriptor if relevant (e.g., colour, art style).
- For retro kits, include the year if identifiable (e.g., "Retro Kit 1953").
- **Example:** `Manchester City FC Graffiti Mural Pattern`
- **Example:** `Liverpool FC Crest Black`

### Player-Specific Kits
- **Format:** `[Full Club Name] [Year] [Player Name] Kit`
- Use the player's commonly known name.
- **Example:** `FC Barcelona 2025 Gavi Kit`

### US College / University
- **Format:** `[University Name] [Mascot Name] [Design Type]`
- Always include the team mascot (Nittany Lions, Crimson Tide, Buckeyes, etc.) — fans search by mascot as often as by school.
- **Example:** `Penn State Nittany Lions Banner`
- **Example:** `Ohio State Buckeyes Logo`

### NHL / US Sports Leagues
- **Format:** `[League] [Full Team Name] [Design Type] [Descriptor]`
- Lead with the league abbreviation (NHL, NFL, NBA, MLB).
- Use full team name including city.
- Add a visual descriptor (Jersey Stripe, Logo, etc.).
- **Example:** `NHL Boston Bruins Jersey Stripe`

### Anime / Manga
- **Format:** `[Series Name] [Sub-series if applicable] [Character Full Name / Group Name] [Design Type]`
- Always include the sub-series when it exists (e.g., "Shippuden" for Naruto, "Super" for Dragon Ball).
- Use full character names (e.g., "Itachi Uchiha" not just "Itachi").
- Use established group/faction names (e.g., "Akatsuki").
- Design types: Icon, Key Art, Character Art, Saga.
- **Example:** `Naruto Shippuden Itachi Uchiha`
- **Example:** `Dragon Ball Super Android Saga`

### Entertainment / Comics / WWE
- **Format:** `[Franchise] [Character Full Name] [Design Descriptor]`
- Use the character's most recognized ring name or alias.
- Add a descriptor that hints at the visual (Main Event, Classic, etc.).
- **Example:** `Batman Classic Logo`
- **Example:** `WWE Rey Mysterio Main Event`

### Music Artists
- **Format:** `[Artist Full Name] [Album/Song/Era Name] [Visual Descriptor]`
- Use the album, song, or era the design references.
- Add a colour or mood descriptor if it aids search.
- **Example:** `Billie Eilish Kyoto Blue`

### Licensed Characters / Cartoons
- **Format:** `[Franchise] [Character Name] [Theme/Occasion] [Descriptor]`
- Include the most iconic character name for search (e.g., "Snoopy" for Peanuts).
- Add occasion or theme if applicable (Birthday, Christmas, Halloween).
- **Example:** `Peanuts Snoopy Birthday Surprise`

---

## Checklist Before Finalizing a Name

- [ ] Is the brand/club/franchise name fully spelled out?
- [ ] Does every word contribute to searchability?
- [ ] Would a customer searching on Amazon/Google type these words?
- [ ] Is it 3–6 words long?
- [ ] Does it follow the category format above?
- [ ] No internal codes, abbreviations, or jargon?

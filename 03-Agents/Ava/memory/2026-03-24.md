## Research: Flash Model Comparison + NemoClaw + Starter Kits (8:30 AM)

### GPT-5.4 Mini & Nano (released Mar 17)
- **GPT-5.4 nano:** $0.20/1M in, $1.25/1M out — designed for "classification, data extraction, supporting subagent tasks"
- **GPT-5.4 mini:** $0.75/1M in, $4.50/1M out — coding agents, sub-agent orchestration, 400K context
- **Both FREE via Codex OAuth** (ChatGPT Plus subscription) — just uses quota
- nano available in OpenClaw as of 2026.3.22 (needs `openai-codex/gpt-5.4-nano`)

### Gemini Flash Models (also free via OAuth)
- **Gemini 3.1 Flash Lite:** $0.25/1M in, $1.50/1M out — fastest TTFT (2.5x faster), 363 tok/s, 1M context
- **Gemini 3 Flash:** $0.50/1M in, $3.00/1M out — 78% SWE-bench, 1M context, stronger reasoning

### Key Insight: Embedding Model ≠ Routing Model
- Gemini Embedding 2.0 handles memory RETRIEVAL (vector search) — this is fixed regardless of routing model
- The routing LLM reads the retrieved results and classifies intent — any fast model works
- Question is: which LLM is best at reading retrieved memory context and making good routing decisions?

### NemoClaw Research Summary
- **What it is:** NVIDIA's enterprise wrapper around OpenClaw — announced GTC 2026 (Mar 17)
- **OpenShell:** Open-source security runtime, YAML-based policy controls (file access, network, API calls)

- **Privacy Router:** Keeps sensitive data local, routes to cloud models only when needed
- **Partners:** Cisco, CrowdStrike, Google, Microsoft Security, Box, Atlassian, Salesforce, SAP
- **Key quote (Jensen Huang):** "Every SaaS company will become a GaaS company" (Generative-as-a-Service)
- **Agent permissions mirror employee permissions** — lock down agents like you lock down users
- **Status:** Early access / alpha preview

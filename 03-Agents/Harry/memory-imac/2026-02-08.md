### Exec Approvals Lesson Learned (IMPORTANT)
- `clawdbot approvals allowlist add --node "X"` modifies a file on the GATEWAY, not the node
- The node reads `~/.clawdbot/exec-approvals.json` LOCALLY on boot
- To fix node approvals: edit the file ON THE NODE directly, then restart node
- Correct schema:
```json
{
  "version": 1,
  "defaults": { "security": "full", "ask": "off", "askFallback": "full" },
  "agents": { "main": { "security": "full", "ask": "off", "askFallback": "full", "allowlist": [{"pattern": "*"}] } }
}
```

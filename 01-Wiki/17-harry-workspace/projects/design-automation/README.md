# Design Workflow Automation Project

**Goal:** Automate the manual design replication process for PlayStation gaming controller skins.

## Two Workflows

### 1. Product Image Generation (`/product-images/`)
- **Owner:** Creative Team
- **Purpose:** Generate marketplace listing images (multiview mockups)
- **Input:** Designer's RAW file (flat artwork)
- **Output:** Multi-angle product shots for Amazon/eBay listings
- **Current Process:** Manual element transfer → Script generates views

### 2. Print Image Creation (`/print-images/`)
- **Owner:** Operations / Production
- **Purpose:** Create .EPS files for manufacturing
- **Input:** Cut file from product image workflow
- **Output:** Print-ready .EPS files
- **Current Process:** Drag cut file → Vecras template → .EPS export

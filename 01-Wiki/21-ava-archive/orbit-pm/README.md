# Orbit — Project Management App
## Technical Specification

### 1. Vision
A beautiful, feature-rich project management tool combining the visual elegance of Linear with the flexibility of Notion.

### 2. Tech Stack
*   **Framework:** Next.js 14 (App Router)
*   **Language:** TypeScript
*   **Styling:** Tailwind CSS + Framer Motion (animations)
*   **UI Library:** shadcn/ui (Radix Primitives)
*   **Database:** PostgreSQL (via Supabase or Neon)
*   **ORM:** Prisma
*   **State:** Zustand (client) + React Query (server state)
*   **Calendar:** FullCalendar or React-Big-Calendar

### 3. Core Features
*   **Workspaces:** Multi-tenant support.
*   **Views:**
    *   **Kanban Board:** Drag-and-drop tasks (dnd-kit).
    *   **List View:** Spreadsheet-like editing.
    *   **Calendar View:** Monthly/Weekly project timelines.
*   **Tasks:** Rich text descriptions, subtasks, tags, priorities.
*   **Team:** Assignees, comments, real-time presence (optional).

### 4. Data Model (Schema Preview)
*   `Workspace` -> `Project` -> `Task`
*   `User` -> `WorkspaceMember`

### 5. Design System ("Nebula")
*   **Font:** Inter (UI) + JetBrains Mono (Code).
*   **Theme:** Dark mode first, deep indigo/violet accents.
*   **Vibe:** Fast, fluid, keyboard-centric.

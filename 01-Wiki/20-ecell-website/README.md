# Design Automation Project

## Two-Track Architecture

### Track 1: Speed to Market (AI-Driven)
Photo/flat design → AI compositing → ecommerce-ready images
- Gemini-driven mockup generation
- Background removal + device compositing
- Seconds per image, fully automated

### Track 2: Print Ready (Async Photoshop)
Automated batch process → CMYK print files
- No smart objects
- UXP scripting
- Runs overnight on iMac

## Folder Structure
- `Gemini_Scripts/` — Cem's Gemini prompts and scripts
- `Source_Files/` — PSDs, raw designs, artwork
- `Templates/` — Device mockup templates
- `Output_Samples/` — Example outputs for QA
- `Reference/` — Competitor examples, inspiration

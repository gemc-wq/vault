## 📐 DESIGN PRINCIPLES FOR ANTIGRAVITY

### 1. Typography
**Headlines:**
- Font: Inter or SF Pro Display
- Weight: 600-700 (SemiBold to Bold)
- Size Scale: 48px (Hero), 36px (H1), 28px (H2), 22px (H3)

**Body Text:**
- Font: Inter
- Weight: 400 (Regular)
- Size: 16-18px
- Line Height: 1.6

**B2B Professional Tone:**
> "Clean geometric sans-serif typography with excellent readability, generous line spacing, and clear hierarchy for busy professionals"

### 2. Layout & Spacing
**Grid:**
- 12-column responsive grid
- Max container width: 1280px
- Gutter: 24px
- Section padding: 80px vertical

**Whitespace:**
- Generous margins (Apple-inspired)
- Card padding: 32px
- Component spacing: 24px

**Antigravity Prompt:**
> "Layout with generous whitespace like Apple, clear visual hierarchy, professional 12-column grid system, ample breathing room for B2B readability"

### 3. Photography Style
**Product Shots:**
- Clean white/light gray backgrounds
- 3/4 angle showing depth
- Macro detail shots (material texture)
- Scale reference (hand holding device)

**Corporate Lifestyle:**
- Professional office environments
- IT departments deploying devices
- Corporate gift presentations
- Manufacturing facility (clean, modern)

**Avoid:**
- Consumer lifestyle (beach, party, casual)
- Overly staged influencer content
- Cheap stock photography

**Antigravity Prompt:**
> "High-quality professional photography with clean backgrounds, 3/4 product angles showing depth and protection, corporate office environments for lifestyle shots, manufacturing facility showcasing precision"

## 🎯 ECCELL B2B POSITIONING: STYLE BALANCE

### The Challenge
Position Ecell between:
- **Consumer brands** (Casetify, Otterbox) → Too playful
- **Promotional suppliers** (4imprint, Vistaprint) → Too cheap
- **Premium B2B** (Apple) → Too expensive

### The Solution
**"Enterprise-grade protection at strategic value"**

**Visual Balance:**
- Professional like Apple (clean, minimal)
- Bold like Otterbox (strong presence)
- Accessible like promotional suppliers (value)
- Premium like Casetify (quality feel)

## 📝 ANTIGRAVITY DESIGN PROMPTS

### Prompt 1: Homepage Hero
```
Create a B2B homepage hero section for Ecell, a device protection manufacturer. 
Style: Clean minimalism like Apple, professional navy blue (#1E3A5F) primary with 
electric blue (#007AFF) accents. Generous whitespace. Large headline typography 
(Inter font, 48px bold). Subheadline (20px regular). Single product showcase 
image of phone case on right. CTA button in gold (#D4A853). Background: white 
with subtle light gray (#F5F7FA) gradient. Corporate, trustworthy, innovative. 
Not playful, not cheap.
```

### Prompt 2: Solutions Page
```
Design a B2B solutions page with three columns: IT Asset Protection, 
Corporate Gifting, License Partnerships. Style: Professional grid layout, 
icon + headline + description + link structure. Icons: Simple line style, 
electric blue (#007AFF). Cards: White background, subtle shadow, 32px padding. 
Hover state: Slight lift + border color change. Typography: 28px section headers, 
18px body. Clean, scannable for busy professionals.
```

### Prompt 3: Product Catalog
```
Create a B2B product catalog page showing phone cases, iPad cases, laptop cases. 
Filter sidebar on left (collapsible on mobile). Product grid: 3 columns desktop, 
2 tablet, 1 mobile. Product cards: Clean image, product name, starting price, 
"Request Quote" button. Minimal information density. White cards on light gray 
background (#F5F7FA). Professional photography style. Easy bulk ordering emphasis.
```

### Prompt 4: Quote Request Form
```
Design a multi-step quote request form for B2B customers. Step indicator at top 
(1-2-3-4). Clean form fields with clear labels. Progress bar. Each step on 
separate "page" with slide transition. Colors: Navy headers (#1E3A5F), white 
background, electric blue accents (#007AFF) for active states. Gold (#D4A853) 
for primary CTA. Form validation with clear error states. Mobile-optimized.
```

### Prompt 5: Trust Signals Section
```
Create a trust signals section for B2B website. Stats row: 4M+ cases manufactured, 
300+ devices supported, 20+ years experience, 50+ license partners. Large numbers 
(48px bold), descriptive labels (14px uppercase). Below: Client logos in grayscale, 
equal height (60px). Below that: Certifications badges. Clean horizontal layout, 
plenty of whitespace. Professional credibility focus.
```

**Overall Vibe:**
- **Corporate** but not stuffy

- Apple's clean minimalism + blue palette

# Royalty Reporting

## Purpose
Dedicated project folder for royalty reporting automation, separate from fulfillment and shipping workflows.

## Scope
- Legacy Zero royalty logic discovery
- Royalty contract and rate model
- Legacy-format royalty export extraction
- Config-driven royalty converter integration
- One-click batch royalty report generation
- Validation, parity testing, and phased modernization

## Current documents
- `ROYALTY_CALCULATION.md` — calculation logic, contract/rate model, and automation SOP
- `LEGACY_LOGIC_EXTRACTION.md` — legacy PHP logic extraction summary
- `IMPLEMENTATION_MAP.md` — target system design and rollout plan
